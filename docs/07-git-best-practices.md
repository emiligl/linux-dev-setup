# Git Best Practices

## Purpose
Define a consistent Git workflow.

## Workflow

- Create feature branches.
- Commit frequently.
- Open Pull Requests.
- Squash when merging.
- Delete merged branches.

## Commit Style

Examples:

```text
Add Grafana dashboard

Fix CloudWatch exporter bug

Update documentation
```
