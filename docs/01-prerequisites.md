# Prerequisites

## Purpose
Define the requirements before configuring the workstation.

## Prerequisites
- Linux distribution
- Bash
- sudo privileges
- Internet access
- GitHub account
- Visual Studio Code (recommended)

## Tested Environment
- CentOS Stream 9
- WSL2

## Best Practices
- Keep the OS updated.
- Use a non-root user for development whenever possible.
